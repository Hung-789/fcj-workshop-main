During my internship at **Amazon Web Services Vietnam** from **05/05/2026** to **20/07/2026**, I had the opportunity to learn, practice, and apply the knowledge acquired in school to a real-world cloud engineering environment.

I proposed and self-selected the topic **AI Agent-based Network Security Monitoring System on AWS** — a subject I had no prior hands-on experience with, requiring me to self-study Amazon Bedrock Agent, generative AI orchestration, and IAM security design from scratch. Through this project, I improved my skills in **AWS networking (VPC, Security Groups, VPC Flow Logs), serverless architecture (Lambda, DynamoDB, S3, SNS, SQS), generative AI application development (Amazon Bedrock Agent, Action Groups, Guardrails), IAM security design, and technical documentation**.

In terms of work ethic, I strived to complete tasks well and complied with workplace regulations. During the Well-Architected review phase, I proactively identified and remediated several security gaps (encryption, IAM least-privilege, reliability) beyond the initial project scope. However, I also made avoidable operational mistakes during testing — including a manual S3 command that unintentionally re-triggered the ingestion pipeline — which highlighted the need for more careful verification before running commands against a live environment.
